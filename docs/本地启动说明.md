# 本地启动说明

本文档记录本项目在本机环境中的启动、后台运行、停止和日志查看方式。

## 1. 进入项目目录

```bash
cd /Users/ianqian/self/aiagents-stock
```

## 2. 首次准备环境

如果项目内还没有 `.venv`，先创建虚拟环境：

```bash
python3 -m venv .venv
```

安装依赖：

```bash
.venv/bin/python -m pip install -r requirements.txt
```

`requirements.txt` 已包含 `streamlit` 和 `schedule`，通常不需要单独安装。

## 3. 前台启动

```bash
.venv/bin/streamlit run app.py
```

启动后访问：

```text
http://localhost:8501
```

如需固定本地地址和端口：

```bash
.venv/bin/streamlit run app.py --server.address 127.0.0.1 --server.port 8501
```

## 4. 后台启动

```bash
nohup .venv/bin/streamlit run app.py --server.headless true --server.address 127.0.0.1 --server.port 8501 > /tmp/aiagents-stock-streamlit.log 2>&1 &
```

查看日志：

```bash
tail -80 /tmp/aiagents-stock-streamlit.log
```

验证服务是否可访问：

```bash
curl -I http://127.0.0.1:8501
```

返回 `200 OK` 表示启动成功。

## 5. 停止服务

查找进程：

```bash
pgrep -fl 'streamlit run app.py'
```

停止进程：

```bash
kill <PID>
```

停止后复查：

```bash
pgrep -fl 'streamlit run app.py'
curl -I --max-time 3 http://127.0.0.1:8501
```

如果没有进程，且端口不可连接，表示已停止。

## 6. 本次验证记录

- 启动命令：`.venv/bin/streamlit run app.py --server.headless true --server.address 127.0.0.1 --server.port 8501`
- 验证地址：`http://127.0.0.1:8501`
- 验证结果：HTTP `200 OK`
- 日志路径：`/tmp/aiagents-stock-streamlit.log`
- 验证后已停止 Streamlit 进程
